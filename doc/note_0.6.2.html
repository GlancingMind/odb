Overview

This is the first release of the ODB project. A simple extension of Ada 95 has been done to incorperate the object persistency into Ada 95.

  • The object data is stored in an XML based file.
  • Small scale transaction management.
  • A simple extension of the Ada 95 to incorperate persistence in the lagnuage
  • Basic migration scenarios are covered, e.g. deletion and addition of attributes are supported
  • Supports object hirarchies (subsutiability).

Since the documentation is still quite poor, a nice example called "Dungons of Wisdom" has been added to demonstrate the usage of the concept in the context of an interactive application. The example is located in the directory ./samples/dungon.

./samples - Some general examples

./test - Basic tests for several ODB features

Restrictions

  • With the intorduction of the XML based format of the object files this or higher version are not compatible any more with any earlier version of ODB. There is no migration path between 0.6.0 and earlier version.

Installation on Linux/Unix

Unpacking and Configuration

$ gunzip -c odb-src-<version>.tar.gz | tar xvf -
$ cd odb-src-<version>
$ ./configure
